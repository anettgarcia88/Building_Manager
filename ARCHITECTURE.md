```sql
CREATE TABLE user_building_roles (
    id UUID PRIMARY KEY,
    userId UUID NOT NULL REFERENCES users(id),
```

## Ejemplos de Consultas SQL

### Obtener todas las relaciones de un usuario
```sql
SELECT ubr.*, b.name as building_name
FROM user_building_roles ubr
JOIN buildings b ON ubr.buildingId = b.id
WHERE ubr.userId = 'user-id-123';
```

### Obtener todos los usuarios de un edificio
```sql
SELECT ubr.*, u.email
FROM user_building_roles ubr
JOIN users u ON ubr.userId = u.id
WHERE ubr.buildingId = 'building-id-456';
```

### Cambiar rol de usuario en edificio
```sql
UPDATE user_building_roles
SET role = 'MANAGER', updatedAt = CURRENT_TIMESTAMP
WHERE userId = 'user-id' AND buildingId = 'building-id';
```

### Eliminar acceso de usuario a edificio
```sql
DELETE FROM user_building_roles
WHERE userId = 'user-id' AND buildingId = 'building-id';
```
